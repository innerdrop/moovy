# Prompts de Cowork — Moovy

Estos son los prompts activos que usamos para trabajar con Opus 4.6 en Cowork.

## Archivos

| Archivo | Cuándo usarlo | Frecuencia |
|---------|--------------|------------|
| `PROMPT_1_INICIAL.md` | Para generar CLAUDE.md y PROJECT_STATUS.md desde cero | Una sola vez (o si hay que regenerar) |
| `PROMPT_2_DIARIO.md` | Para cada sesión de trabajo en Cowork | Todos los días |

## Cómo usar

1. Abrí Cowork → seleccioná la carpeta de moovy
2. Copiá el contenido del prompt que corresponda
3. Pegalo en Cowork
4. Dejá que Opus trabaje

## Notas

- El Prompt 1 ya se ejecutó el [completar fecha]. No hace falta volver a ejecutarlo salvo que quieras regenerar todo desde cero.
- El Prompt 2 es el de uso diario. Siempre el mismo. Opus lee CLAUDE.md y PROJECT_STATUS.md y sigue donde quedó.
